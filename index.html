Defination:
In React, props is short for "properties" and is a way to pass data from a parent component to a child component. The parent component can pass data to the child component by adding attributes to the child component's JSX element, and the child component can access that data through the props object. The props object is read-only, meaning the child component cannot modify the data passed down to it via props.

Short Note:
useState is a built-in hook in React that allows functional components to have state and re-render when that state changes. It takes an initial value as an argument and returns an array with two elements: the current state value and a function to update that value.

Example:
When you use useState in a functional component in React, you need to pass an initial value to it as an argument. For example, let's say you want to track a count in your component, and you want to initialize it to 0

  1. The first element is the current state value, which we've assigned to the count variable using array destructuring. Initially, count is set to 0.
  2. The second element is a function that allows us to update the state value. We've assigned this function to setCount. We can call setCount with a new value to update the count state. In the example above, we use setCount(count + 1) to increment the count by 1 whenever the "Increment" button is clicked.
